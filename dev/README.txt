# README - Dossier développement (dev)

Ce dossier contient les éléments nécessaires au développement, à la maintenance et à l'évolution de l'application Word Atelier.

Inclure dans ce dossier :
- scripts/ : scripts de build, scraping, enrichissement de données, etc.
- logs/ : fichiers de log, rapports de debug, etc.
- progression-word.json : fichier de progression de test ou de développement

Ne pas inclure dans la version distribuable (app/) :
- scripts/
- logs/
- progression-word.json

Pour toute modification ou évolution, travailler dans ce dossier puis générer la version finale dans app/.
